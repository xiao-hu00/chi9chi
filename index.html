    # 序言

    前端知识体系十分庞大，驳杂且难以整理，网上也有各种知识架构图，看新手看了就会感觉头晕。

    入行也有好几年了，前面一段时间有好几个人问我如何入门，我也只能甩给对方一堆网站，当然也是有用的，不过对于新手不大友好，而完整又大略覆盖整个前端体系的文章有比较少，不好推荐，鉴于此，虽然本人技术深度还不够，还是想分享一些自己理解的前端知识给初级入门的人。

    本文的定位的读者是尚未入门的前端开发者。

    # HTML

    说起前端基础，那么HTML是绕不过去的坎。HTML历史发展略过，我们只需要知道它是做什么的就可以了。

    HTML是整个网页结构的骨架，所有的内容都处于这个骨架之中，打开谷歌浏览器（其他浏览器）的开发者模式(F12、mac:cmd+Alt+i)，可查看相关功能，前端开发几乎所有的效果都在浏览器上实现，所以掌握浏览器的调试功能是一项必备技能。

    作为新手通常有个疑惑，就是看到一个页面不知道从何下手，我觉得是因为新手有时候想的太多，我们拿到一个页面的设计图，首先不必去管各项功能的实现，我们首先要做的只是搭建出骨架，其实就是写HTML，不用考虑css或者js逻辑，第一步先把页面搭建起来。
